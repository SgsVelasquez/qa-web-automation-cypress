\# QA Web Automação - Cypress



Projeto de automação de testes E2E utilizando Cypress em aplicação web pública.



\## 🔧 Tecnologias utilizadas



\- Cypress

\- JavaScript

\- Node.js

\- Git/GitHub



\## Aplicação testada



https://www.saucedemo.com/



\## Cenários de teste



\- Login com sucesso

\- Login com credenciais inválidas

\- Adição de produto ao carrinho



\## Como executar



Instalar dependências:



```bash

npm install

```



Abrir Cypress:



npx cypress open



Rodar em modo headless:



npx cypress run



\## Objetivo



Demonstrar automação de testes E2E em aplicação web, validando fluxo de autenticação, tratamento de erros e interação com elementos da interface.
